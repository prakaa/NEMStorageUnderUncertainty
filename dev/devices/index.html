Summary

Initialises a battery energy storage system (BESS).

throughput (in MWh) can be supplied in cases where the BESS has already undertaken energy storage and discharge. This is akin to cycling but is independent of storage capacity (significant where calendar and/or cycling degradation is accounted for). If not supplied, default value is 0.0.

Returns

A BESS
